# eco-city-manager
Proyecto de Programación Orientada a Objetos: Simulación de una ciudad sustentable (Eco-City).

# Sistema de Distribución CEDIS
Este proyecto simula un sistema de distribución de productos utilizando POO.
“El sistema está desarrollado con Programación Orientada a Objetos utilizando las siguientes clases:”

Producto → controla nombre y stock
Inventario → gestiona los productos
Vendedor → genera pedidos aleatorios
CEDIS → coordina todo el sistema

## Funcionalidades
- Gestión de inventario
- Generación de pedidos
- Control de stock
- Simulación de varios días

## Objetivo
Analizar el comportamiento del inventario y prevenir desabasto.
